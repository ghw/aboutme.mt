6.3. 安装 Xorg

Xorg 是由 X.Org 基金会发行的 X Window 系统的开源实现。 在 FreeBSD 中它可以通过预编译包或 Ports Collection 安装。 Ports Collection 中有一套完整的 X 服务器， 包括服务器端、 客户端、 库 （libraries）和 字体等等， 可以通过 x11/xorg 安装。 同时有个最小化的 Xorg 软件可以通过 x11/xorg-minimal 来安装，有独立的 ports 可用于安装 文档， 库 （libraries） 和应用程序。 本章示例为安装完整 Xorg 软件套件。

从 Ports Collection 编译安装 Xorg ：

# cd /usr/ports/x11/xorg
# make install clean

Note:

编译完整的 Xorg 软件套件需要至少 4 GB 磁盘可用空间。

或者， 直接通过预编译包安装 Xorg ， 通过 pkg_add 命令安装：

# pkg_add -r xorg

通过 pkg 命令安装：

# pkg install xorg
