7.4. 文档查看器

从 UNIX® 出现至今， 一些新的文档格式开始流行，浏览它们的阅读器基本系统里不一定包含 本节中，将介绍如何安装以下文档查看器：

软件名称系统资源需求从 Ports 安装耗时主要依赖
Acrobat Reader®低短Linux® 兼容层
gv低短Xaw3d
Xpdf低短FreeType
GQview低短Gtk+ 或 GNOME

7.4.1. Acrobat Reader®

现在许多文档发布时都采用便携式文件格式便攜式文件格式 (PDF) 。 有个主流的 PDF 查看器叫 Acrobat Reader® ， 它是 Adobe® 针对 Linux® 操作系统发布的。使用 Linux® 兼容层， FreeBSD 也能使用它。 由于授权许可证限制， 它没有预编译包可用， 这意味着这个软件必须从 ports 编译安装， 有几个本地语言版可用在 Ports Collection 里的 print 目录 (freebsd.org/ports/print.html) 下找到。

以下命令为从 Ports Collection 安装 Acrobat Reader® 9 英文版本， 如要改为本地语言版本， 则需要 cd 到所需语言版本的 Port 目录， 编译安装。

# cd /usr/ports/print/acroread9
# make install clean

7.4.2. gv

gv 是 PostScript® 和 PDF 格式的文档查看器。 它源自 ghostview 因为采用 Xaw3d 引擎， 外观看起来更美观。 gv 有很多功能， 比如：方向性、 纸张大小、 缩放和抗锯齿等。 而且几乎所有功能都可以只用键盘或鼠标完成。

通过预编译包安装 gv 输入：

# pkg_add -r gv

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/print/gv
# make install clean

7.4.3. Xpdf

对于那些喜欢小型 PDF 查看器的用户， FreeBSD 提供了轻量级， 高效率， 资源需求小的 Xpdf 文档查看器。 它使用标准的 X 字体， 不需要任何额外的工具包。

安装 Xpdf 预编译包，使用：

# pkg_add -r xpdf

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/graphics/xpdf
# make install clean

一旦完成安装，就可用运行 xpdf ，使用鼠标中键激活选择菜单。

7.4.4. GQview

GQview 是一个图片管理器。 您可以单击鼠标来查看一个文件、打开外部编辑器和 预览功能。 它也有幻灯片播放模式和一些基本的文件操作。 您可以管理采集的图片并且很容易找到重复的。 GQview 可以全屏幕观看并且支持国际化。

通过预编译包安装 GQview ，输入：

# pkg_add -r gqview

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/graphics/gqview
# make install clean
