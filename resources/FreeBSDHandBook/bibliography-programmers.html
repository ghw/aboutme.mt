B.4. 开发方面的参考文献

  • 加州大学伯克利分校的计算机研究小组的 4.4BSD 程序员参考手册(4.4BSD Programmer's Reference Manual) ， 由奥莱利公司 (O'Reilly & Associates, Inc) 于 1994 年出版发行， ISBN 1-56592-078-3。

  • 加州大学伯克利分校的计算机研究小组的 4.4BSD Programmer's Supplementary Documents， 由奥莱利公司 ( O'Reilly & Associates, Inc) 于 1994 年出版发行， ISBN 1-56592-079-1。

  • Harbison, Samuel P 和 Steele, Guy L. Jr 的 C 语言参考手册(C: A Reference Manual) 第四版， 由 Prentice Hall 于 1995 年出版发行， ISBN 0-13-326224-3。

  • Kernighan, Brian 和 Dennis M. Ritchie 的 The C Programming Language 第二版 由 Prentice Hall 于 1988 年出版发行， ISBN 0-13-110362-8。

  • Lehey, Greg 的 Porting UNIX Software， 由奥莱利公司 (O'Reilly & Associates, Inc.) 于 1995 年出版发行， ISBN 1-56592-126-7。

  • Plauger, P. J 的 The Standard C Library 由 Prentice Hall 于 1992 年出版发行， ISBN 0-13-131509-9。

  • 斯平内利斯 (Spinellis, Diomidis) 的 代码阅读方法与实践(Code Reading: The Open Source Perspective) 由艾迪生韦斯利 (Addison-Wesley) 于 2003 年出版发行， ISBN 0-201-79940-5。

  • 斯平内利斯 (Spinellis, Diomidis) 的 Code Quality: The Open Source Perspective 由艾迪生韦斯利 (Addison-Wesley) 于 2006 年出版发行， ISBN 0-321-16607-8。

  • 史蒂文斯 (Stevens, W. Richard) 和拉戈 (Stephen A. Rago) 的 UNIX环境高级编程 (Advanced Programming in the UNIX Environment) 第二版， 由马萨诸塞州： 艾迪生韦斯利(Addison-Wesley) 于 2005 年出版发行， ISBN 0-201-43307-9。

  • 史蒂文斯 (Stevens, W. Richard) 的 UNIX网络编程(UNIX Network Programming) 第二版， 由 Prentice Hall 于 1998 年出版发行， ISBN 0-13-490012-X。
