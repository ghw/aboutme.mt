7.3. 生产力套件

当需要进行办公时， 新用户通常都会选择一个好用的办公套件或字处理软件。 尽管目前有有些 桌面环境 ， 如 KDE 已经提供了办公套件， 但目前这还没有一定之规。 无论您使用哪种窗口管理器， FreeBSD 都能提供您需要的软件。

本节演示如何安装以下常见软件，并从应用程序的系统资源需求、 从 ports 编译安装耗时，依赖情况，这些方面做简要分析。

软件名称系统资源需求从 Ports 安装耗时主要依赖
Calligra低短KDE
AbiWord低短Gtk+ 或 GNOME
The Gimp低长Gtk+
Apache OpenOffice高长JDK™ 和 Mozilla
LibreOffice较高长Gtk+ 或 KDE/ GNOME 或 JDK™

7.3.1. Calligra

KDE 桌面环境包含一个办公套件， 它可以从 KDE 单独安装。 Calligra 包含其它办公套件中能找到的标准组件。 Words 是字处理程序、 Sheets 是电子表格程序、 Stage 是演示文档制作管理程序、 Karbon 是矢量绘图软件。

在 FreeBSD 中， editors/calligra 可以通过预编译包或 port 安装。通过预编译包安装：

# pkg_add -r calligra

如果没有预编译包可用，可用使用 Ports Collection 编译安装：

# cd /usr/ports/editors/calligra
# make install clean

7.3.2. AbiWord

AbiWord 是一个外观跟 Microsoft® Word 很类似的免费的字处理程序， 它非常快且包含许多特性，并且非常容易使用。

AbiWord 可以导入或导出多种文件格式， 包括 Microsoft® .rtf 格式。

使用预编译包安装 AbiWord ：

# pkg_add -r abiword

如果没有预编译包可用， 可以从 Ports Collection 编译安装：

# cd /usr/ports/editors/abiword
# make install clean

7.3.3. GIMP

对于图片的编辑和修改来说， GIMP 是个优秀的图形处理软件。 它可以作为一个简单的绘图程序或者一个专业的照片处理套件。 它支持大量的插件和具有脚本界面。 GIMP 可以读写众多的文件格式， 并支持扫描仪和手写板。

通过预编译包安装：

# pkg_add -r gimp

或者， 使用 Ports Collection 安装：

# cd /usr/ports/graphics/gimp
# make install clean

Ports Collection 里的 graphics 目录下(freebsd.org/ports/graphics.html) 有几个 GIMP 相关的插件， 帮助文件和用户手册。

7.3.4. Apache OpenOffice

Apache OpenOffice 是一款开源的办公套件， 是 Apache 软件基金会的孵化器项目之一。 它是一套完整的办公套件， 包括：字处理程序、 电子表格程序、 演示文档管理程序和绘图程序。 它有其他办公套件类似的外观， 还能导入导出几种常见的文件格式。它支持多种语言的用户界面、 拼写检查和字典。

Apache OpenOffice 字处理使用 XML 文件格式来增加可移植性和灵活性。 电子表格程序支持宏语言和使用外部数据源。Apache OpenOffice 可以稳定的运行在 Windows® 、 Solaris™ 、 Linux® 、 FreeBSD 和 Mac OS® X 等操作系统下。更多有关 Apache OpenOffice 的信息可以在 openoffice.org 网站找到。 与 FreeBSD 相关的信息可以在 porting.openoffice.org/freebsd/ 网站上找到。

安装 Apache OpenOffice 预编译包可使用：

# pkg_add -r apache-openoffice

一旦完成安装，就可以输入以下命令启动 Apache OpenOffice ：

% openoffice-X.Y.Z

其中 X.Y.Z 是安装的 Apache OpenOffice 的版本号。 首次启动时， Apache OpenOffice 会询问一些问题， 并会在用户主目录中创建 .openoffice.org 文件夹。

如果没有所需的 Apache OpenOffice 预编译包，那么从 port 编译安装也是一种选择。 但是这需要大量的空闲磁盘空间和相当长的编译时间：

# cd /usr/ports/editors/openoffice-4
# make install clean

Note:

如果需要编译本地语言版， 只需使用以下命令替换之前的命令：

# make LOCALIZED_LANG=your_language install clean

您必须把 your_language 替换成您需要的语言， port 所支持的 ISO-code 可以在这个 port 的 files/Makefile.localized 文件中找到。

7.3.5. LibreOffice

LibreOffice 是 documentfoundation.org 开发的一款免费的办公套件。 它与主流的办公套件与各种平台兼容，它是 OpenOffice.org 的衍生版本，并包含一套完整办公套件： 一个字处理程序、 电子表格程序、 演示文档管理程序、 绘图程序、 数据库管理程序以及用于创建和编辑数学公式的 æ 程序。

LibreOffice 字处理使用 XML 文件格式来增加可移植性和灵活性。 电子表格程序支持宏语言和使用外部数据源。LibreOffice 可以稳定的运行在 Windows® 、 Linux® 、 FreeBSD 和 Mac OS® X 操作系统之上。有关 LibreOffice 的更多信息请访问 libreoffice.org 网站。

需要安装英文版 LibreOffice 预编译包，使用：

# pkg_add -r libreoffice

Ports Collection 中的 editors 目录下 (freebsd.org/ports/editors.html) 有几个本地语言版 LibreOffice ， 如果想安装本地语言版需要将 libreoffice 替换为本地语言版的软件包名称。

一旦安装完成，就可以输入以下命令运行 LibreOffice：

% libreoffice

首次启动时， 会询问一些问题， 并会在用户主目录中创建 .libreoffice 目录。

如果没有所需的 LibreOffice 预编译包，那么从 port 编译安装仍然是一种选择。 这需要大量的空闲磁盘空间和相当长的编译时间。 以下使用编译英文版本的 LibreOffice 为例子：

# cd /usr/ports/editors/libreoffice
# make install clean

Note:

如果需要编译本地语言版本，可以 cd 到所需语言版本的 Port 目录， 编译安装。 支持何种本地语言可以在 Ports Collection 的 editors 目录下(freebsd.org/ports/editors.html) 找到。
