7.5. 财务

对于想在 FreeBSD 桌面上管理个人财务信息， 有些强大和易于使用的应用程序可以安装。 它们中的兼容一些常见的文件格式，比如： Quicken 和 Excel 格式。

本节介绍以下软件：

软件名称系统资源需求从 Ports 安装耗时主要依赖
GnuCash低长GNOME
Gnumeric低长GNOME
KMyMoney低长KDE

7.5.1. GnuCash

GnuCash 是 GNOME 致力于为最终用户提供用户友好且功能强大的软件的一部分。 使用 GnuCash ， 您可以关注您的收入和开支、您的银行帐户， 或者您的股票。 它的外观看起来非常的专业。

GnuCash 提供一个智能化的注册、帐户分级系统、 很多键盘快捷方式和自动完成方式。 它能分开一个单个的处理到几个详细的部分。 GnuCash 能导入和合并 Quicken QIF 格式文件。 它也支持大部分的国际日期和流行的格式。

通过预编译包安装 GnuCash ，输入：

# pkg_add -r gnucash

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/finance/gnucash
# make install clean

7.5.2. Gnumeric

Gnumeric 是 GNOME 社区开发的电子表格程序。 它以通过元素格式和许多片断的自动填充系统来方便的自动“猜测”用户输入而著称。 它能导入一些流行的文件格式，比如像 Excel 、 Lotus 1-2-3 或 Quattro Pro 。 它有大量的内置函数和常见的单元格格式， 如数字、 货币、 日期、 时间等等。

通过预编译包安装 Gnumeric ，输入：

# pkg_add -r gnumeric

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/math/gnumeric
# make install clean

7.5.3. KMyMoney

KMyMoney 是 KDE 创建的个人理财软件。 KMyMoney 旨在提供并融合各种商业财务管理软件所有的重要特性。 它也同样注重易用性和特有的复式记帐功能。KMyMoney 能从标准的 Quicken QIF 文件中导入数据， 追踪投资，处理多种货币并能提供一个财务报告。

通过预编译包安装 KMyMoney ，输入：

# pkg_add -r kmymoney-kde4

如果预编译包不可用，则可用使用 Ports Collection 编译安装：

# cd /usr/ports/finance/kmymoney-kde4
# make install clean
